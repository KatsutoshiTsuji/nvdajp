Fichero Readme para NVDA

Un lector de pantalla de código abierto (y libre) para el sistema operativo Windows.

Creado por Michael Curran, con la ayuda de James Teh y otros.

Acceso al escritorio de no videntes (NVDA en sus iniciales en inglés) es un lector de pantalla de código abierto para el sistema operativo Windows, que capacita a la gente ciega y con discapacidad visual para utilizar sus ordenadores por un costo no mayor que el del ordenador y del sistema operativo en sí. Iniciado en abril del 2006, ha crecido hasta llegar a ser bastante usable como un lector de pantalla para el día a día, capacitando al usuario para hacer la mayoría de las tareas que necesitase. No es tan estable o libre de fallos como algunos de los lectores de pantalla comerciales. Pero desde  diciembre del 2006 el creador de NVDA ha sido capaz de utilizar NVDA a tiempo completo como su lector de pantalla principal, abandonando finalmente su producto comercial original. 

Proporcionando el apoyo de un sintetizador de voz, NVDA permite al usuario acceder y interactuar con todas las partes del sistema operativo Windows, tales como: 

Navegar por la web (con Internet Explorer y Firefox)
Leer y escribir documentos con programas tales como WordPad o Microsoft Word
Enviar y recibir correo electrónico con Outlook Express 
Utilizar programas de línea de órdenes en ventanas de Dos
Producir hojas de cálculo básicas en Microsoft Excel
Administradción general del ordenador a través Mi PC / Explorador de Windows , elementos del Panel de Control, y otros Tareas genéricas de Windows

NVDA no está limitado por una necesidad para seguir tendencias y demandas del mercado. Los lectores de pantalla comerciales implementan características realmente buenas, pero algunas veces esto está predispuesto por qué quiere el mercado, Antes que si serán realmente útiles para el usuario. NVDA podría no ser siempre tan estable como otros lectores de pantalla, pero ciertamente puede actuar como una erramienta experimental y de prueba. Puede probar fácil y rápidamente nuevas ideas y características quizás no vistas en otros lectores de pantalla para el sistema operativo Windows. Un ejemplo de esto es la capacidad del NVDA para utilizar pitidos para comunicar al usuario que una barra de progreso se está moviendo. El pitido más alto, el final de la barra de progreso. La gente ha añadido esta característica en otros lectores de pantalla a través de scripts personalizados y similares, pero nunca ha sido oficialmente adaptado completamente en el núcleo de ningún otro lector de pantalla para Windows. 

Puedes ayudar en el desarrollo de NVDA comunicándo fallos cuando los encuentres. Las instrucciones para informar fallos, y un número de artículos útiles, pueden ser encontrados en el wiki en: www.nvaccess.org/nvda/wiki. Or, si quisieras más información general acerca de NVDA puedes visitar el sitio Web en: www.nvaccess.org/nvda. Allí, puedes encontrar también información sobre unión a listas de correo de NVDA. Actualmente hay dos listas: una para discusiones relacionadas con los usuarios en general y una lista de desarrolladores para quienes ejecutan y prueban NVDA desde el código fuente.
