      Ознакомительный файл "NVDA" 

      Свободная, с открытым исходным кодом, программа речевого доступа к экрану под Windows.

      Разработчик Michael Curran, с помощью James Teh и др.

      "NVDA" (Свободная программа речевого доступа к экрану под Windows с открытым исходным кодом) позволяет незрячим и слабовидящим людям использовать компьютер без дополнительных затрат на приобретение специальных программ. "NVDA" развивается с начала апреля 2006 года и становится довольно успешной программой экранного доступа для повседневного использования. Она не лишена недостатков, и, пока, менее функциональна, чем некоторые комерческие программы речевого доступа к экрану, но с декабря 2006 года, разработчик "NVDA" использует её в качестве основного средства экранного доступа, оставив приобретённый прежде комерческий продукт. 

С помощью вывода информации синтезатором речи, "NVDA" позволяет получить доступ к большинству приложений Windows, например,просматривать Интернет, используя "Internet Explorer" и "Firefox";
читать и редактировать документы в "WordPad" или "Microsoft Word";
отправлять и получать электронную почту через "Outlook Express"; 
использовать программы через командную строку в Dos окнах;
создавать простые таблицы в "Microsoft Excel";
осуществлять общее обслуживание компьютера через приложения "Мой Компьютер" (My Computer) / "Проводник" (Windows Explorer(), компоненты "Панели Управления" (Control Panel applets) и выполнять другие стандартные задачи под Windows. 

NVDA не связана ограничениями и тенденциями комерческого рынка. Комерческие программы экранного доступа обладают рядом достоинств, однако, многие их функции, встроенные по требованиям рынка, подчас, не несут практического расширения возможностей пользователя по реализации равного доступа. Возможно, "NVDA", иногда, уступает в стабильности некоторым программам экранного доступа, но может использоваться как эксперементальный и исследовательский инструмент. Его можно применять для опробирования и внедрения функций, не встречающихся в других программах речевого доступа к экрану под Windows. Простым примером служит использование тональных сигналов "NVDA", показывающих изменение состояния индикаторов выполнения, чем выше том, тем ближе к концу процесс. Пользователи других продуктов экранного доступа добавляют подобную функциональность с помощью специальных макросов, однако, официально, она нигде полностью не интегрирована в ядро программ.

            Меню интерфейса NVDA: Здесь вы можете 
настраивать параметры NVDA и, также, можете завершить работу программы. 
Для вызова меню интерфейса в любое время, нажмите insert+n.
Для выхода из меню  интерфейса, без завершения работы NVDA, нажмите Esc. 
 
      Для завершения работы NVDA, в любое время,
нажмите insert+q или выберите Выход из меню интерфейса NVDA. 
Появится диалог с вопросом о завершении работы программы и вы сможете нажать кнопку OK или Отмена. 
 
      Для изменения таких параметров, как: настройки Голоса, Эхо Клавиатуры, Чтение Подсказок, 
Используйте insert+n для активации меню интерфейса NVDA, затем, клавиши со стрелками для перемещения по пунктам меню. 
Нажатие клавиши Ввод на большинстве пунктов данного меню вызовет появление диалогов 
параметров, где вы сможете изменить индивидуальные настройки. Многие 
настройки применяются сразу, например, изменение скорости голоса активного 
синтезатора и т.д., поэтому, вы легко 
сможете подобрать желаемые значения. 
Однако, если вы нажмёте кнопку Отмена и закроете диалог, сделанные вами 
изменения настроек не будут приняты и значения параметров вернутся к 
предыдущему состоянию.  
 
      Поумолчанию, настройки сохраняются только на текущую сессию работы NVDA. Чтобы сохранить настройки конфигурации,  выберите "Сохранить Конфигурацию" из меню интерфейса NVDA. Вы можетеотметить флажок, "Автосохранение конфигурации при выходе". Откройте Общие Настройки ... из меню, Параметры и, нажимая клавишу Таб, найдите флажок "Автосохранение конфигурации при выходе". Нажмите клавишу Пробел, затем, ok.

NVDA+Страница вниз - Уменьшить скорость речи 
NVDA+p - Включает и выключает чтение пунктуации

NVDA+f - Читать параметры текущего шрифта в документе
