NonVisual Desktop Access (NVDA)
URL: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) é un lector de pantalla de código aberto (e libre) para o Sistema Operativo Windows. Este programa pode capacitar á xente cega ou deficiente visual ó acceder a ordenadores executando Windows por un coste non maior que una persoa vidente. NVDA permite ó usuario coñecér que está a pasar na pantalla inquerindo ó Sistema Operativo e utilizando  un sintetizador de voz para a información de saída.

Para máis información, por favor mira a guía do usuario incluída. Pódense atopar información adicional e recursos no Sitio Web do NVDA.
