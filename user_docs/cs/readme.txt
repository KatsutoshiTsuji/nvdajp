NonVisual Desktop Access (NVDA)
Web: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) je otevřený (a volně šiřitelný) odečítač obrazovky pro operační systém Windows.

Potřebné informace najdete nejen v uživatelské příručce, ale také na webových stránkách programu NVDA (viz výše) a nebo na stránce českých a slovenských fanoušků projektu NVDA na adrese:
http://nvda.venik.org
