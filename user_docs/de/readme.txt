NonVisual Desktop Access (NVDA)
Internet: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) ist ein OpenSource Screen Reader für Windows, der es blinden und sehbehinderten Menschen ermöglicht, ihren Computer ohne zusätzliche Kosten nutzen zu können. NVDA erlaubt dem Benutzer herauszufinden, was am Bildschirmpassiert, indem es das Betriebssystem danach abfragt und eine Sprachausgabe zur Informationsausgabe nutzt.

Für weitere Informationen lesen Sie das beiliegende Benutzerhandbuch. Auf der Webseite von NVDA finden Sie ebenfalls weitere Informationen.
