NonVisual Desktop Access (NVDA)
Internetseite: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) ist ein quelloffener Bildschirmleser für Windows-Betriebssysteme, der es blinden und sehbehinderten Menschen ermöglicht, ihren Computer ohne zusätzliche Kosten zu benutzen. NVDA erlaubt dem Benutzer herauszufinden, was am Bildschirmpassiert, indem es das Betriebssystem danach abfragt und eine Sprachausgabe zur Inforationsausgabe nutzt.

Für mehr Informationen lesen Sie das beiliegende Benutzerhandbuch. Auf der Webseite von NVDA findenSieweitere Informationen.
